# Book Search App

## How to install
- Install Python 3.5+ https://www.python.org/downloads/
- Install pip https://pip.pypa.io/en/latest/installing/
- Install Django
```bash
$pip install django
```
- Install Node Package Manager (npm) https://nodejs.org/en/download/
- In folder /backend
```bash
$python manage.py runserver
```
- In folder /frontend
```bash 
$npm install
$npm start
```
